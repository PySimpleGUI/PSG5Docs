## Reminder about project closure

The PySimpleGUI project is closing at the end of the year.  The servers are going off-line and the product support will end.  The documentation will be hosted here on GitHub so that Commercial License holders will be able to continue to reference it.  

A final release of PySimpleGUI 5 is being prepared.  Information on how Commercial License holders can obtain it will be emailed to Commercial Licenses registered email address.
