    User Screenshots

    The user screenshots are an automatically generated gallery view of issue 10 in the PySimpleGUI GitHub repository.

    https://github.com/PySimpleGUI/PySimpleGUI/issues/10

    Post screenshots of your programs created with PySimpleGUI in the issue above to have it appear here!

    The plain view only shows images posted on the issue while the captioned view shows other text and comments in the issue.

    Note: there may be posts of images that are not PySimpleGUI programs. Please try your best to only post screenshots of PySimpleGUI programs in issue 10. If you have a question/bug/request, open a separate issue.
